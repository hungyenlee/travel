# 台灣散步筆記

以台灣各縣市為單位，整理景點與美食店家的旅遊資訊網站。第一版聚焦台南，其他縣市保留入口。

## Language

**地點（Place）**：
網站收錄的一個單位，是「景點」或「美食店家」其中一種。兩者共用同一套資料結構，用 `type` 欄位區分。
_Avoid_: 點位、item、location（location 保留給經緯度欄位）

**景點（Attraction）**：
以參觀、遊覽為主的地點，`type` 為 `attraction`。例如赤崁樓、安平古堡。
_Avoid_: 觀光點、spot

**美食店家（Restaurant）**：
提供餐飲的實際店家，`type` 為 `restaurant`。第一版直接以實際店家為資料單位，不另做食物種類介紹。
_Avoid_: 餐廳、店、shop、美食（「美食」保留作分類/導覽用詞，不指單一店家）

**城市 slug（City slug）**：
每個縣市固定的英文識別碼，用於網址參數與檔名，例如 `tainan`、`kaohsiung`、`taichung`、`taipei`、`hualien`。需搭配「中文顯示名」對照。
_Avoid_: city id、城市代碼

**識別碼（id）**：
每個地點唯一的英文識別碼，用於網址參數，例如 `restaurant-001`、`chihkan-tower`。

## 角色

**遊客（Visitor）**：
瀏覽網站的人。仍無帳號、資料永不離開瀏覽器、無伺服器搜尋。唯一的輸入場景是「行程規劃」——一個純本機（localStorage）的互動：命名行程、把地點分配到「第 N 天」。除此之外只能點選由允許清單產生的篩選按鈕，沒有其他輸入框（連搜尋都沒有）。
_Avoid_: 使用者（「使用者」太籠統，會和維護者混淆）、user

**維護者（Maintainer）**：
維護內容的人，第一版即網站作者本人。維護方式是手動編輯 `data.js`。清單外標籤的頁面橫幅提示是給維護者看的安全網，與遊客無關。
_Avoid_: 管理者（管理者保留給未來的登入後編輯功能）、admin

## 行程規劃

**行程（Trip）**：
遊客建立的命名清單，收集想去的地點並分配到「第 N 天」。存於瀏覽器 localStorage、可同時有多個。第一版可跨城市（雖然目前只有台南）。
_Avoid_: 計畫、plan（頁面檔名用 `plan.html`，但中文一律講「行程」）

**第 N 天（Day）**：
行程裡的抽象天數桶，用來安排地點大致落在旅程的第幾天，**不綁真實日期**。`day` 值：`0` = 未分配、`1..N` = 第 1..N 天。移除中間某天後其餘天自動連號重排，永遠 1..N 連續。
_Avoid_: 日期、date（第一版不做真實日期）

**未分配（Unassigned）**：
尚未決定哪天去的地點所在處，即 `day` 為 `0`。地點加入行程時預設落在這裡。
_Avoid_: 待排、待定

**加入行程（Add to trip）**：
從詳細頁或城市頁列表卡，把一個地點放進某個行程的動作。點按鈕後彈出選單，可 toggle 加入／移除，或就地建立新行程。（首頁精選卡、地圖預覽卡不放此按鈕。）
_Avoid_: 收藏、加入最愛、favorite

**拖曳排序（Drag to reorder）**：
在規劃層用卡片左側的「握把 `⋮⋮`」把地點卡拖到別的位置或別的天——同一天內調整先後、或搬到另一天／未分配。序號＝該天內順序，與地圖編號標記一致。桌機用滑鼠、手機長按握把啟動；「⋯」選單的「移到某天／上移／下移」是等效的非拖曳路徑。
_Avoid_: 排程、schedule（這裡只排先後與天，不涉真實時間）

## 分類與標籤

**允許清單（Taxonomy）**：
一份預先維護的合法用詞清單，篩選按鈕只從這份清單產生。地點的分類與標籤必須從清單挑選；填了清單外的詞時，網站會在畫面上明顯提示。涵蓋「行政區、景點分類、料理分類、特色標籤」四類，不涵蓋招牌品項。
_Avoid_: 主清單、字典

**行政區（District）**：
地點所在的市轄區，例如中西區、安平區。每個城市各有一份行政區清單。
_Avoid_: 區域、area

**分類（Category）**：
說明地點的性質。景點用「景點分類」（古蹟、博物館⋯⋯），店家用「料理分類」（牛肉湯、台式小吃⋯⋯）。屬於允許清單。
_Avoid_: 類型、kind

**特色標籤（Tag）**：
描述地點的情境或特色，例如老店、可外帶、宵夜。屬於允許清單，可篩選。
_Avoid_: 標記、label

**招牌品項（Signature dish）**：
店家的具體招牌菜名，例如溫體牛肉湯、牛肉燥飯。自由填寫，不屬於允許清單、不做篩選。
_Avoid_: 菜色、menu item
